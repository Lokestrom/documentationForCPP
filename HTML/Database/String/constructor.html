String documentation

constructor

constexpr String() noexcept; (1)
String(const String& s) noexcept; (2)
constexpr String(const char* s) noexcept; (3)
String(Vector<char> v) noexcept; (4)
String(const std::string& s) noexcept; (5)


(1) Default constructor. Constructs a empty string with unspecified capacity.
(2) Copy constructor. Constructs a String identical to the input String.
(3) C string constructor. Constructs a String with all characters
from the char* pointer to the first '\0' character.
(4) Vector constructor. Constructs a string from a Vector of chars.
(5) std::string copy constructor. copies a std::string.

Complexity

(1) constant.
(2) linear in s.size().
(3) linear in length between pointer and nearest '\0' character.
(4) linear in v.size().
(5) linear in s.length().
