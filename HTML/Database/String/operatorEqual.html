operator=

constexpr operator= (const String& s) noexcept ; (1)
constexpr operator= (const char* s) noexcept ; (2)
operator= (const Vector <char >& v) noexcept ; (3)
constexpr operator= (const std::string& s) noexcept ; (4)
constexpr operator= (const std::initializer_list <char >& iList) noexcept ; (5)


(1) Replaces the characters of the String to the characters of the String s.
(2) Replaces the characters of the String to the characters between the
char* s pointer and first '/0' character.
(3) Replaces the characters of the String to the characters stored in the Vector.
(4) Replaces the characters of the String to the characters of the std::string s.
(5) Replaces the characters of the String to the characters in the
std::initializer_list<char> iList.

Complexity

(1) linear in s.length().
(2) linear in length between char* s pointer and nearest '\0' character.
(3) linear in v.size().
(4) linear in s.length().
(5) linear in iList.size().
