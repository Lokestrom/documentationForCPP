String documentation

operator=

String& operator=(const String& s) noexcept; (1)
constexpr String& operator=(const char* s) noexcept; (2)
String& operator=(const std::string& s); (3)
String& operator=(const Vector<char>& v); (4)
String& operator=(const std::initializer_list<char>& iList) noexcept; (5)


(1) Copy constructor. Constructs a String identical to the input String.
(2) Assignes a C string to the String. Assignes all characters
from the char* pointer to the first '\0' character.
(3) Assignes a std::string to the String.
(4) Assignes the character from a Vector<char> to the String.
(5) Assignes the character from a std::initializer_list<char> to the String.

Complexity

(1) linear in s.size().
(2) linear in length between pointer and nearest '\0' character.
(3) linear in s.size().
(4) linear in v.length().
(5) linear in iList.length().
